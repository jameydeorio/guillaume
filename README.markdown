# Guillaume

Guillaume is a generative poetry creation tool.

## Installation

Add this line to your application's Gemfile:

    gem 'guillaume'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install guillaume

## Usage

    $ ruby -Ilib bin/guillaume
